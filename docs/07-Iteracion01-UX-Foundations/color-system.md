# Color System

## Primary

Blue

Used for:
- Navigation
- Primary Actions
- Links

## Success

Green

Used for:
- Completed Priorities
- Positive CRS
- Success States

## Warning

Amber

Used for:
- Risks
- Carry-Overs
- Attention States

## Danger

Red

Used for:
- Blockers
- Errors
- Critical States

## Neutral

Gray

Used for:
- Borders
- Backgrounds
- Secondary Information

## CRS Color Mapping

90-100 -> Green
80-89  -> Blue
70-79  -> Amber
<70    -> Red
