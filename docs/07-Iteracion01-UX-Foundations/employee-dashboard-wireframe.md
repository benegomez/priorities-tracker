# Employee Dashboard Wireframe

## Objective

Provide employees with immediate visibility into commitments, progress, risks, and upcoming actions.

## Layout

Header
- Logo
- Current Planning Cycle
- Notifications
- Profile

My CRS
- Current CRS
- Trend
- Reliability Category

Upcoming Actions
- Pending Check-In
- Pending Check-Out

My Priorities
- Active
- Completed
- Carry-Over

Risks & Blockers
- Open Risks
- Open Blockers

Recent Activity

## Primary Actions

- Create Check-In
- Create Check-Out
- View Priorities
- Update Progress

## Success Criteria

Employee understands:
- What am I committed to?
- What remains pending?
- What requires action?
