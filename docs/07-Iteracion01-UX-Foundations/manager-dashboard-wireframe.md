# Manager Dashboard Wireframe

## Objective

Provide managers with immediate visibility into team execution health.

## Layout

Header

Team Health KPIs
- Check-In %
- Check-Out %
- Average CRS
- Risks
- Blockers

Team Activity
- Active Priorities
- Completed Priorities
- Carry-Overs

CRS Trends
- Team Reliability Graph

Risks & Blockers
- Open Risks
- Critical Risks
- Blockers

AI Weekly Summary

Team Members

## Primary Actions

- View Employee
- Review Risks
- Review CRS
- Open Reports

## Success Criteria

Manager understands:
- Team Status
- Execution Health
- Emerging Risks
