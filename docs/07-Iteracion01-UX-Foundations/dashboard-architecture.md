# Dashboard Architecture

## Dashboard Philosophy

Dashboards are action-oriented.

Users should immediately understand:

- What requires attention
- What is blocked
- What is at risk
- What needs action

---

## Employee Dashboard

### My Priorities

- Active Priorities
- Completed Priorities
- Pending Priorities

### Upcoming Actions

- Pending Check-In
- Pending Check-Out

### Reliability

- Current CRS
- Historical Trend

### Risks

- Open Risks
- Open Blockers

---

## Manager Dashboard

### Team Health

- Check-In Completion
- Check-Out Completion
- Average CRS

### Team Activity

- Active Priorities
- Completed Priorities
- Carry-Overs

### Risks

- Open Risks
- Critical Risks
- Blockers

### Insights

- AI Summary
- Execution Trends
- CRS Trends
