Use Cases

Caso de Uso 1: Planeación Semanal

Actor Principal

Empleado

Flujo

1. Inicia semana.
2. Ingresa al sistema.
3. Selecciona proyectos.
4. Selecciona fases.
5. Recupera prioridades pendientes.
6. Define nuevas prioridades.
7. Registra tareas.
8. Envía Check-In.

Resultado

Manager obtiene visibilidad inmediata de los compromisos.

⸻

Caso de Uso 2: Seguimiento de Riesgos

Actor Principal

Empleado

Flujo

1. Detecta un bloqueador.
2. Registra el riesgo.
3. Manager recibe visibilidad.
4. Manager toma acciones.

Resultado

Se reduce la probabilidad de retrasos.

⸻

Caso de Uso 3: Cierre Semanal

Actor Principal

Empleado

Flujo

1. Revisa prioridades.
2. Marca tareas completadas.
3. Marca prioridades completadas.
4. Registra comentarios.
5. Define elementos que continúan.
6. Envía Check-Out.

Resultado

Se genera historial de cumplimiento.

⸻

Caso de Uso 4: Preparación de Reunión 1:1

Actor Principal

Manager

Flujo

1. Revisa historial del colaborador.
2. Consulta prioridades activas.
3. Consulta bloqueadores.
4. Revisa CRS.
5. Realiza reunión.

Resultado

Conversaciones enfocadas en resultados y apoyo.

⸻

Caso de Uso 5: Evaluación de Confiabilidad

Actor Principal

Manager

Flujo

1. Consulta histórico de cumplimiento.
2. Analiza CRS.
3. Detecta tendencias.
4. Identifica oportunidades de coaching.

Resultado

Evaluación objetiva basada en compromisos cumplidos.
