User Stories

Administrador

Gestión Organizacional

Como administrador,

Quiero crear proyectos y fases,

Para que los colaboradores puedan asociar correctamente sus prioridades.

⸻

Como administrador,

Quiero gestionar equipos y usuarios,

Para mantener la estructura organizacional actualizada.

⸻

Como administrador,

Quiero asignar managers a equipos,

Para garantizar visibilidad adecuada de los compromisos.

⸻

Empleado

Check-In Semanal

Como empleado,

Quiero registrar mis prioridades de la semana,

Para comunicar claramente mis compromisos.

⸻

Como empleado,

Quiero reutilizar prioridades pendientes de semanas anteriores,

Para evitar capturar información repetida.

⸻

Como empleado,

Quiero registrar tareas específicas para cada prioridad,

Para descomponer mis compromisos en actividades concretas.

⸻

Seguimiento

Como empleado,

Quiero registrar bloqueadores,

Para que mi manager pueda ayudarme oportunamente.

⸻

Check-Out

Como empleado,

Quiero indicar qué prioridades completé,

Para reflejar mis resultados reales.

⸻

Como empleado,

Quiero trasladar tareas pendientes a la siguiente semana,

Para mantener continuidad en mi trabajo.

⸻

Manager

Seguimiento

Como manager,

Quiero visualizar todas las prioridades activas de mi equipo,

Para entender rápidamente en qué están trabajando.

⸻

Como manager,

Quiero identificar prioridades sin avance,

Para intervenir antes de que se conviertan en retrasos.

⸻

Como manager,

Quiero revisar bloqueadores reportados,

Para ayudar a resolverlos.

⸻

Desempeño

Como manager,

Quiero consultar el CRS de cada colaborador,

Para entender patrones de cumplimiento.

⸻

Como manager,

Quiero analizar tendencias históricas,

Para detectar mejoras o deterioros en la ejecución.
