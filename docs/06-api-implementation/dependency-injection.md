Centralizar construcción de dependencias.

## Componentes

- Database Session
- Repositories
- Services
- AI Gateway

## Ejemplo

Router
 ↓
Use Case
 ↓

## Beneficios

- Testabilidad

- Sustitución sencilla de implementaciones
