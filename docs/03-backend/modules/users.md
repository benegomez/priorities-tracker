# Users Module

## Objetivo
Gestionar usuarios de la plataforma.

## Responsabilidades
- Crear usuarios
- Actualizar perfil
- Activar / desactivar usuarios

## Entidades
- User

## Casos de Uso
- CreateUserUseCase
- UpdateUserUseCase
- DeactivateUserUseCase

## Reglas
- Todo usuario pertenece a una organización.
