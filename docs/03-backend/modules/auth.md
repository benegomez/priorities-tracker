# Auth Module

## Objetivo
Gestionar autenticación y autorización.

## Responsabilidades
- Login
- Logout
- JWT
- Refresh Tokens (futuro)
- Gestión de permisos

## Casos de Uso
- LoginUserUseCase
- ValidateTokenUseCase

## Entidades
- AuthSession
- Role

## Dependencias
- Users Module
- Shared Security
