# CRS Formula

## Fórmula Base

CRS = (
40% Prioridades Completadas +
30% Tareas Completadas +
20% Consistencia Histórica +
10% Penalización por Arrastre
)

## Componentes

### Prioridades
prioridades_completadas / prioridades_comprometidas

### Tareas
tareas_completadas / tareas_comprometidas

### Consistencia
Promedio móvil de últimas 8 semanas.

### Arrastre
Penalización por prioridades continuadas.

## Escala

90-100 = Excelente
75-89 = Confiable
60-74 = Riesgo Moderado
0-59 = Riesgo Alto
