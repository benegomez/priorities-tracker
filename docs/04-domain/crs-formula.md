# CRS Formula v1.0

## Objetivo

Medir la confiabilidad de cumplimiento de compromisos.

## Componentes

1. Prioridades Completadas (40%)
2. Tareas Completadas (30%)
3. Consistencia Histórica (20%)
4. Penalización por Arrastre (10%)

## Fórmula Conceptual

CRS =
(0.40 * cumplimiento_prioridades)
+ (0.30 * cumplimiento_tareas)
+ (0.20 * consistencia_historica)
+ (0.10 * factor_arrastre)

## Escala

90-100 Excelente
75-89 Confiable
60-74 Riesgo Moderado
0-59 Riesgo Alto

## Tendencias

- Improving
- Stable
- Declining

## Versionado

CRS v1.0 será almacenado junto con cada cálculo para permitir evolución futura.
