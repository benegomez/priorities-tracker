# Domain Scenarios

## Escenario 1: Check-In Semanal

1. Empleado selecciona Proyecto.
2. Selecciona Fase.
3. Define Prioridades.
4. Define Tareas.
5. Completa Check-In.

## Escenario 2: Continuidad

1. Check-Out identifica prioridad incompleta.
2. Empleado marca continuidad.
3. Prioridad aparece disponible en siguiente Check-In.

## Escenario 3: Cálculo CRS

1. Check-Out completado.
2. CRS Engine ejecuta cálculo.
3. Reporting actualiza métricas.
