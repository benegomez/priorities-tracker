# Domain Events

## CheckInCompleted
Se dispara al finalizar un Check-In.

## CheckOutCompleted
Se dispara al finalizar un Check-Out.

## PriorityCarriedOver
Se dispara cuando una prioridad continúa a otra semana.

## CRSCalculated
Se dispara al recalcular CRS.

## ProjectPhaseCompleted
Se dispara al cerrar una fase.
