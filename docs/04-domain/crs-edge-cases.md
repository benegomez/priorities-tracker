# CRS Edge Cases

## Empleado Nuevo

Sin historial.

Regla:
- Consistencia histórica neutral.

## Semana Sin Actividad

Sin Check-In.

Regla:
- No generar CRS.

## Sobrecompromiso

Muchas prioridades, bajo cumplimiento.

Regla:
- Penalización indirecta por cumplimiento.

## Arrastres Recurrentes

Prioridad arrastrada múltiples semanas.

Regla:
- Incrementar penalización.

## Prioridades Parciales

Prioridad incompleta con tareas completadas.

Regla:
- Prioridad no completada.
- Tareas sí cuentan para score parcial.
